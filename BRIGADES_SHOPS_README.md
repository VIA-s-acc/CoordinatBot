# Управление бригадами и магазинами

Этот файл описывает 2 способа добавления направлений (бригад и магазинов):

1. Через Telegram-бот (рекомендуется)
2. Вручную через конфиг `data/brigades_shops.json`

---

## 1) Добавление прямо в боте

Доступно только для роли `SUPER_ADMIN`.

### Путь в меню
- `📋 Меню`
- `👥 Управление ролями`
- `🏗️🏪 Управление бригадами/магазинами`

### Что можно делать
- `➕ Бригада` — добавить новую бригаду
- `➕ Магазин` — добавить новый магазин
- `📋 Бригады` / `📋 Магазины` — посмотреть список
- `✏️ Редактировать бригаду` / `✏️ Редактировать магазин` — изменить поля (`name`, `spreadsheet_id`, `sheet_name`, `owner_id`)
- `🗑 Удалить бригаду` / `🗑 Удалить магазин` — удалить запись

### Шаги добавления
Бот запросит по очереди:
1. `name` — название сущности
2. `spreadsheet_id` — ID Google таблицы
3. `sheet_name` — имя листа в таблице
4. `owner_id` — Telegram user id владельца (или `-`, если без владельца)

После подтверждения запись сразу сохраняется в `data/brigades_shops.json`.

### Удобные значения `-`
- На шаге `spreadsheet_id`: можно отправить `-`, тогда будет взят default из `.env`.
- На шаге `sheet_name`: можно отправить `-`, тогда бот возьмет default из `.env`, а если он не задан — имя сущности.

---

## 1.1) Автоматизация через `.env`

Можно задать стандартные значения для новых сущностей:

```env
DEFAULT_BRIGADE_SPREADSHEET_ID=1AbCdEf...
DEFAULT_SHOP_SPREADSHEET_ID=1XyZkLm...

DEFAULT_BRIGADE_SHEET_NAME=
DEFAULT_SHOP_SHEET_NAME=
```

Примечание:
- Если `DEFAULT_*_SHEET_NAME` пустой, лист будет называться как сама сущность (`name`).
- При первой записи, если листа с таким именем нет, бот создаст его автоматически.

## 2) Ручное добавление через конфиг

Файл: `data/brigades_shops.json`

### Структура

```json
{
  "brigades": [
    {
      "name": "Бригада А",
      "spreadsheet_id": "1AbCdEfGh...",
      "sheet_name": "Расходы_Бригада_А",
      "owner_id": 123456789
    }
  ],
  "shops": [
    {
      "name": "Магазин Центр",
      "spreadsheet_id": "1XyZkLmNo...",
      "sheet_name": "Расходы_Магазин_Центр",
      "owner_id": null
    }
  ]
}
```

### Поля
- `name` (string) — отображаемое название
- `spreadsheet_id` (string) — Google Spreadsheet ID
- `sheet_name` (string) — имя листа
- `owner_id` (number или `null`) — Telegram ID владельца

### Важные правила
- Названия внутри одного типа (`brigades` или `shops`) должны быть уникальными.
- `spreadsheet_id` и `sheet_name` обязательны.
- После ручного изменения сохраните JSON в UTF-8.

---

## Проверка после добавления

1. Откройте в боте меню `💸 Ծախս`.
2. Выберите тип (`🏗️ Բրիգադ` или `🏪 Խանութ`).
3. Убедитесь, что новая сущность отображается в списке.
4. Сделайте тестовую запись и проверьте попадание в нужный лист.
